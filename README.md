Dieses Skript (`happygay.pl`) fuegt dem Irssi "ein paar" neue Befehle
hinzu, die allesamt (ausser `/happygay_about`) mit einem Nickname
aufgerufen werden und dann diverse Sprueche klopfen;
`/happygay_winker` beispielsweise:

`* Daniel winkt °*:.{- Dave -}.:*° mal fröhlich zu. "Juhuuuuuuuuu, Dave!" :-)))`

der Rest ist auch so harmlos ;)

Perl-Teile sind allesamt von mir)
